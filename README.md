## Objective

The objective of this task is to create an automated **data preprocessing pipeline** using Python libraries such as **Pandas and Scikit-learn**. The pipeline performs data cleaning, transformation, and preparation for machine learning workflows.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

---

## Dataset

Titanic Dataset

The dataset contains information about passengers such as:

* Passenger class
* Gender
* Age
* Fare
* Survival status

---

## Pipeline Workflow

1. Load dataset using Pandas
2. Handle missing values using SimpleImputer
3. Encode categorical variables using OneHotEncoder
4. Scale numerical features using StandardScaler
5. Combine preprocessing steps using ColumnTransformer
6. Split dataset into training and testing data

---

## Output

* Processed dataset ready for machine learning models
* Training and testing datasets saved for further use

---

## Conclusion

This project demonstrates how automated data pipelines can streamline preprocessing tasks and prepare datasets efficiently for machine learning models.
